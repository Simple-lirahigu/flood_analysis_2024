## 三、对你当前论文的使用建议

你的样本点部分建议这样写逻辑：

1. 洪水样本点来自 2024-06-23 至 2024-07-06 洪灾期 Sentinel-1 提取的洪水范围。
2. 永久水体、高频季节性水体、陡坡、小斑块噪声应剔除。
3. 非洪水样本点应从洪水范围外、永久水体外、洪水边界缓冲区外随机选取。
4. 正负样本数量可以先保持 1:1，后续可用敏感性实验说明样本比例对结果的影响。
5. 可引用“逆频率比采样方法”相关文献，说明普通随机采样可能存在空间偏差，必要时可进行分层或加权采样。
6. 所有样本点必须限制在缩小后的研究区 `roi_xiao` 内。
