# 洪涝易发性研究代码存档

本仓库用于存档 2024 年鄱阳湖/赣北洪灾洪水易发性研究相关脚本、方法记录和文献整理。

## 当前内容

- `scripts/`：GEE 脚本、TIF 检查、样本因子提取、降水栅格对齐等脚本。
- `docs/`：参考文献、CNKI/英文文献候选、方法说明等文档。
- `MEMORY.md`：已确认的方法、数据、参数和重要上下文。
- `HISTORY_LOG.md`：项目工作记录。
- `WORKSPACE_MAP.md`：本地文件说明。

## 注意

- 原始遥感数据、TIF 因子、GEE 导出样本、Zotero 数据库快照和全文提取文件不上传到 GitHub。
- 当前主要 GEE 样本脚本为 `scripts/gee_extract_2024_flood_nonflood_samples_roi_xiao.js`。
- 脚本中的 GEE Asset 路径需要按个人账号替换。
